Query
------------------
==== Examples ====
BANDS: 40,20,10M,70CM,2MM
SENDER: <MYCALL>
RECEIVER: ANY
LIMIT:
MODE: ANY
PAST_HOURS: 24
REFRESH: YES

==== Defaults ====
> BAND_SUFFIXES:
 NONE = M, M, CM, MM
> PAST_HOURS:
 DEFAULT: 3 MAX: 24
> REFRESH:
 DEFAULT: YES, EVERY 1 MIN

* Leave blank for ANY *
